# AIコードレビューアプリ（OpenAI API利用）

## 概要
AI（ChatGPT）がコードレビューを行ってくれるアプリです。  
React＋Vite環境のコードに対して修正箇所や最適化の指摘をしてくれます。  
回答自体もMarkdown形式で回答してくれます。
